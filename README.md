## 🚀 LaslesVPN Landing Page
A modern and responsive landing page for a VPN service built using HTML and CSS. This project is designed to showcase a clean UI, structured layout, and responsive design principles.

## 🌐 Live Demo

👉[https://shamsshahyar6-create.github.io/Landing-page/]()

## 📌 Features

✅ Clean and modern UI design

✅ Fully responsive layout

✅ Navigation bar with sections

✅ Hero section with CTA (Call To Action)

✅ Features section with highlights

✅ Pricing plans section

✅ Customer testimonials

✅ Global network section

✅ Footer with links and social media

## 🛠️ Technologies Used

HTML5

CSS3 (Flexbox)

Font Awesome Icons

Google Fonts
